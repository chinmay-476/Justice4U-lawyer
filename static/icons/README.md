# LegalMatch PWA Icons
# 
# This directory contains the PWA icons for the LegalMatch application.
# 
# Current files:
# - icon-192x192.svg: SVG version of 192x192 icon
# - icon-512x512.svg: SVG version of 512x512 icon
# 
# To convert SVG to PNG:
# 1. Use online converters like https://convertio.co/svg-png/
# 2. Or use command line tools like ImageMagick: convert icon-192x192.svg icon-192x192.png
# 3. Or use browser: Open SVG in browser, right-click, save as PNG
# 
# Required PNG files for PWA:
# - icon-192x192.png (192x192 pixels)
# - icon-512x512.png (512x512 pixels)
# 
# Optional additional icons:
# - shortcut-lawyers.png (96x96 pixels)
# - shortcut-contact.png (96x96 pixels) 
# - shortcut-apply.png (96x96 pixels)
# - screenshot-desktop.png (1280x720 pixels)
# - screenshot-mobile.png (390x844 pixels)
#
# Note: The SVG files are provided as source files. 
# Convert them to PNG format for the PWA to work properly.
